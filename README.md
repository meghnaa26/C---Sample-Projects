# C Programming Projects

A collection of beginner C projects built to 
practice fundamental programming concepts.

## Projects

### 1. Calculator
A calculator that performs +, -, *, /, % operations.
Features: division/modulus by zero handling, 
invalid operation handling, repeat option using 
string comparison (strcmp).

### 2. Number Guessing Game
A game where the computer picks a random number 
(1-100) and the user guesses it.
Features: too high/too low hints, attempt counter, 
play again option using srand() and rand().

### 3. Student Grade Calculator
A program that calculates a student's percentage 
and grade based on marks across multiple subjects.
Features: input validation, dynamic subject count 
using loops, grade classification (A-E), 
pass/fail determination.

## Concepts Practiced
- String comparison with strcmp()
- Boolean logic with stdbool.h
- Random number generation (srand, rand)
- Loops (for, while)
- Conditional statements (if/else if/else)
- Input validation
- Mathematical operations (fmod for float modulus)
- Arrays and strings (char arrays)

## Built With
